# Fenil Patel Portfolio

Static portfolio built with React, Three.js, and browser-side JSX via Babel.

## Files
- `index.html` — main entry point
- `portfolio-app.jsx` — React app logic
- `portfolio-data.jsx` — portfolio content data
- `tweaks-panel.jsx` — UI controls for theme and 3D settings

## Deploy
1. Push the repository to GitHub.
2. Enable GitHub Pages in repo settings with branch `main` and folder `/ (root)`.
3. Wait a minute, then visit:
   - `https://fenilpatel-05.github.io/fenil-patel-portfolio/`

## Notes
- This project is a static site and does not require a build process.
- Keep the files together in the repo root.
